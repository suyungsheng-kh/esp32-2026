Day 2：光影與閘門 —— 進階控制

🎯 專題目標：類比數值變化與伺服馬達系統

接續第一天的燈光基礎，我們將學習連續數值（類比）的控制，包含利用可變電阻調光調色、光敏電阻製作自動路燈。最後加入超音波與伺服馬達，打造一套智慧停車場閘門系統！

模組原理與概念

可變電阻與光敏電阻

類比輸入 (Analog)：能讀取 0~4095 的連續數值。可變電阻像音量旋鈕可手動調整；光敏電阻則會隨環境光線強弱自動改變電阻值。

WS2812 環狀燈

全彩定位燈：只需一條訊號線，就能串聯控制多顆 LED。你可以指定「第幾顆燈」亮「什麼顏色」，非常適合做幸運轉盤或時鐘秒針效果。

伺服馬達與超音波

超音波用聲波反射測量距離。伺服馬達 (Servo)能精準控制旋轉角度 (通常是 0~180 度)，是製作智慧閘門、機器人關節的關鍵零件。

模組小試身手：各別功能測試

1
可調亮度的燈

接線：可變電阻, 全彩LED, 1602A LCD

  • 用可變電阻控制 LED 亮度。
  • 在 LCD 顯示目前的亮度值。
  • 進階：將數值轉換成調色盤 (紅➝綠➝藍 漸變)。
2
智慧路燈

接線：光敏電阻, LED

🎯 任務：讀取光敏電阻數值，模擬天色。設定一個閥值，天黑時自動開燈，天亮時自動關燈。

3
停車場閘門

接線：超音波, 伺服馬達

🎯 任務：超音波偵測到車子靠近 (<10cm) 時，伺服馬達轉動 90 度打開閘門，等待 3 秒後自動放下。

挑戰一：環狀燈 (WS2812)

運用 WS2812 RGB 環形燈製作炫麗特效：

  • (1) 電子時鐘： 讓一顆燈像秒針一樣，每一秒往前跳一格繞圈。
  • (2) 幸運轉盤： 結合按鈕，按下後燈光快速旋轉，放開後隨機停在一個位置！
挑戰二：停車費計算

升級停車場閘門系統：

  • 車子進入後，開始利用「變數」計時。
  • 車子離開時 (超音波再次偵測)，停止計時。
  • 根據停留的秒數計算停車費，並顯示在 1602A LCD 上！

程式邏輯挑戰 (停車場閘門)

「如果『超音波距離』小於 10 公分，表示有車輛靠近。將『伺服馬達』轉到 90 度開啟閘門，等待車輛通過後，再將馬達轉回 0 度關閉閘門。」

如果 (距離 < 10 cm) 馬達角度設為 90 度 (開門) 等待 3 秒 馬達角度設為 0 度 (關門)
虛擬碼 (Pseudo Code) Day 2: 停車閘門 
Loop {
  dist = read_ultrasonic(D7, D8)
  
  if (dist < 10) {
    servo_write(D5, 90)   // 打開閘門
    lcd_print("Welcome!")
    delay(3000)           // 等待車輛通過
  } else {
    servo_write(D5, 0)    // 放下閘門
    lcd_print("Stop")
  }
  
  delay(100)
}
互動模擬器：調整車輛距離測試閘門

距離 < 10cm 時開啟閘門 90 度，車輛通過。

三小時實作通關指南

1
第 1 小時：調光大師與智慧路燈

🎯 任務目標： 完成任務 1 (調光燈與調色) 及任務 2 (智慧路燈)。

  • 讀取可變電阻，並使用 LCD 顯示數值，再將數值對應到 LED 亮度或顏色。
  • 利用光敏電阻控制 LED 開關，找到適當的「天黑閥值」。
2
第 2 小時：炫彩舞台與時間魔法

🎯 任務目標： 突破「挑戰一」，控制 WS2812 環形燈。

  • 了解陣列概念，利用迴圈與變數控制「第 N 顆燈」。
  • 實作電子時鐘(每秒亮下一顆)，或是結合按鈕製作幸運轉盤抽獎遊戲。
3
第 3 小時：智慧停車場閘門與收費 (挑戰二)

🎯 任務目標： 完成任務 3 (超音波+伺服馬達)，並挑戰停車費計算。

  • 結合超音波測距與伺服馬達角度控制，打造自動閘門。
  • 進階： 使用「變數」累加計時，當車輛離開時，在 LCD 印出「Fee: $XX」的停車費用。
